### 📌 About the Project

**StarkTech AI — InternFinder Platform** is an AI-powered web application that helps students discover and apply for internships using smart match scoring and personalized recommendations.

This prototype was built in **56 hours** for **Technothon at Techno India University** by a **team of 6 (5 males, 1 female)** — including me, **Roshan Mallick**.  
The project was started just **2 days after joining college**.  
Although not shortlisted, it demonstrates **rapid prototyping, teamwork, and full-stack fundamentals.**

---

## 🧠 Key Features

- **AI Match Scoring** — Calculates match percentage:
  - 70% skills
  - 15% location
  - 15% work type
- **Real-Time Search** — Predictive suggestions for:
  - Internships
  - Companies
  - Locations
- **Responsive Design**
  - Mobile-first layout
  - Dark/Light theme toggle
- **Profile & Tracking**
  - Manage skills
  - Bookmark listings
  - Track application status
- **Data Persistence**
  - Uses browser `localStorage` — no backend required
