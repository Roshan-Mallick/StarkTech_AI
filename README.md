# StarkTech_AI - InternFinder Platform

> **Prototype built in 56 hours for Technothon at Techno India University**
> **Team of 6 (5 males, 1 female), including me Roshan Mallick**
> **Started project after just 2 days of joining college**
> **Not shortlisted but demonstrates rapid prototyping skills**

An AI-powered web application that helps students discover and apply for internships using a smart scoring algorithm and personalized recommendations.

## Key Features

- **AI Match Scoring**: Calculates match percentage (70% skills, 15% location, 15% work type)
- **Real-Time Search**: Predictive suggestions for internships, companies, and locations
- **Responsive Design**: Mobile-first layout with dark/light theme toggle
- **Profile & Tracking**: Manage skills, bookmark listings, and track application status
- **Data Persistence**: Uses browser localStorage—no backend required

## Tech Stack

- HTML5 & CSS3
- JavaScript (ES6+)
- Client-side AI algorithms

## Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Roshan-Mallick/StarkTech_AI-InternFinder-Platform.git
   ```
2. **Open** `index.html` in your browser.
3. **Complete your profile** to explore AI-driven internship recommendations.
